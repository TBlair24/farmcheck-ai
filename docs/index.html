Project Structure

Five Phases

Built phase-by-phase to match every requirement in RTV's Data Scientist job description.

01
Dataset & Annotation
41k PlantVillage images ingested, stratified to 1,500 samples, annotated via Label Studio with RTV-mapped compliance indicators.
PlantVillage Label Studio DVC 15 Classes
02
Computer Vision Model
YOLOv8n-cls trained with augmentation and class weighting. 93.8% top-1 accuracy tracked across 30 epochs with W&B.
YOLOv8 PyTorch W&B 93.8% Acc
03
Edge Deployment
Model exported to ONNX for cross-platform field deployment. Benchmarked PyTorch vs ONNX latency and accuracy on CPU.
ONNX Runtime ~8ms Latency ARM-ready
04
MLOps & REST API
FastAPI service with Pydantic schemas, Swagger docs, Docker containerisation, GitHub Actions CI/CD, deployed on Render.
FastAPI Docker GitHub Actions Render
05
Data Pipeline & Dashboard
Medallion architecture in DuckDB ingests predictions, enriches with compliance metadata, aggregates household adoption scores.
DuckDB Bronze/Silver/Gold Plotly Dash 50 Households
